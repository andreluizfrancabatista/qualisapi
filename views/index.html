O que é QualisAPI?

O QualisAPI é um recurso que permite consultar o estrato Qualis de um periódico, nas classificações do quadriênio 2013-2016 e a mais recente 2017-2018, por meio de uma busca pelo seu respectivo número ISSN. Obs.: os dados referentes a classificação de 2019, não são oficiais mas sim uma prévia.

Por enquanto apenas buscas por número de ISSN são permitidas, além de filtros por estratos (ex.: 0101-9031 ou 0101-9031/B1). No futuro serão implementadas as buscas por título do periódico.

Essa API utiliza dados públicos do sistema Qualis Periódicos mas não visa o substituir. Pelo contrário, é um recurso extra para aqueles que buscam uma API para estudos, pesquisas, e desenvolvimento de aplicações diversas que porventura necessitam do estrato de periódicos.

Como funciona?

Por meio de endpoints (URLs) o usuário tem acesso ao estrato Qualis de um periódico em todas as áreas de avaliação na qual ele se encontra. Os dados retornados estão em formato JSON. Para ter acesso aos dados de um periódico, por meio desta API, você deve:

  1. Utilizar o número de ISSN do periódico desejado para criar a URL do endpoint;
  2. Acessar o endpoint via URL criada;
  3. Obter os resultados.

Esta API facilita o uso de dados da plataforma Qualis Periódicos para desenvolvimento de outras aplicações, como por exemplo a extensão para o Google Chrome ChromeQualis.
